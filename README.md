# Movie Search App
It is a simple and interactive React Movie Search Application that allows users to search for movies and view essential details instantly. The app uses the OMDb API to fetch real-time movie data and displays results in visually appealing movie cards.

## Features
- Search movies by name
- Real-time data fetching using OMDb API
- Movie cards with poster, title, year & type
- Modern UI with gradient design
- Fully responsive layout
- Fast and lightweight React application

## Tech Stack
- Frontend: React.js (useState Hook)
- API: OMDb API
- Styling: CSS (Flexbox & Gradients)
- Build Tool: Vite

## How It Works
- User enters a movie name in the search box
- On clicking Search, API request is sent to OMDb
- Movie data is fetched and stored in state
- Results are displayed using reusable MovieCard components
